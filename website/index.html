Overview

A unified framework for multi-robot coordination using Gaussian Belief Propagation

This project implements a unified framework for multi-robot trajectory tracking with collision avoidance using Gaussian Belief Propagation. The system supports both traditional tracking and sensing-based adaptive control, making it suitable for real-time multi-robot coordination scenarios.

Our approach leverages distributed optimization through GBP to achieve scalable coordination while maintaining safety constraints and trajectory tracking performance across multiple autonomous agents.
